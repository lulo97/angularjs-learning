AngularJS is a javascript framework

AngularJS extend HTML with directive:
- ng-app: Khai báo vùng áp dụng AngularJS
- ng-model: Liên kết data với các input, select, textarea để thay đổi data
- ng-bind: Liên kết data với tag hiện tại để hiển thị
- ng-init: Khởi tạo một biến trong AngularJS
